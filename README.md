# Primetime app

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

## Build app

1. EAS Build

   ```bash
   eas build -p <platform> --profile <profile>
   ```

2. Submit to apple store

   ```bash
    eas submit
   ```
